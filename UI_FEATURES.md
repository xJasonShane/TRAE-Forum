## 视图切换

页面提供三种视图模式，通过工具栏右侧的视图切换按钮或键盘快捷键 `1` / `2` / `3` 切换：

<table>
<tr>
<td width="33%">

### 📊 分类视图（默认）

- 帖子按分类分组，每分类独立列
- 分类按帖子数量降序
- 列头显示图标、名称、数量
- 置顶帖子显示 📌 标记
- 点击标题新标签页打开

</td>
<td width="33%">

### 🃏 卡片视图

- 卡片网格形式展示
- 显示封面图、标签、标题、摘要
- 无封面图时显示分类图标
- 超过 100 帖子自动懒加载
- 适合浏览详细内容

</td>
<td width="33%">

### 📅 日历视图

- 月历形式展示帖子分布
- 有帖子日期显示色点和数量
- 点击日期展开帖子列表
- 支持月份前后导航
- 今日日期高亮

</td>
</tr>
</table>

## 排序功能

| 排序方式 | 说明 |
|:---------|:-----|
| 🕐 最新发布 | 按发布时间倒序，最新的帖子在前 |
| 🕐 最早发布 | 按发布时间正序，最早的帖子在前 |
| 👁️ 最多浏览 | 按浏览量排序，浏览量高的在前 |
| ❤️ 最多点赞 | 按点赞数排序，点赞多的在前 |
| 💬 最多回复 | 按回复数排序，回复多的在前 |

排序偏好自动保存到 localStorage 和 URL。

## 搜索功能

**搜索范围：** 帖子标题 · 帖子摘要/内容 · 分类名称

| 特性 | 说明 |
|:-----|:-----|
| ⚡ 实时搜索 | 输入后 200ms 自动搜索（防抖优化） |
| 🔢 结果计数 | 显示匹配的帖子数量 |
| ✖️ 一键清除 | 点击 × 按钮清除搜索内容 |
| 🔗 分类联动 | 搜索结果更新分类标签的计数 |
| 🌐 URL 同步 | 搜索词同步到 URL hash，支持分享 |

## 深色模式

- 使用 CSS 变量实现主题切换，覆盖所有 UI 元素
- 点击头部月亮/太阳图标或按 `D` 键切换
- 自动保存偏好到 localStorage
- 支持 URL 参数 `#theme=dark` 直接进入深色模式
- 深色主题下所有颜色、阴影、边框均有独立变量定义

---

## 数据统计

- 点击头部「统计」按钮或按 `S` 键打开统计面板

| 图表 | 说明 |
|:-----|:-----|
| 🍩 分类分布图 | 环形图展示各分类帖子占比（Chart.js） |
| 📊 发帖时间线 | 柱状图展示每月发帖数量趋势 |

- 图表颜色与分类配置颜色一致
- 再次点击或按 `Esc` 关闭面板

---

## 数据导出

- 点击头部「导出」按钮展开下拉菜单

| 格式 | 说明 |
|:-----|:-----|
| 📄 CSV | 包含 ID、标题、分类、创建时间、浏览、点赞、回复、链接；自动添加 BOM 头确保中文编码正确 |
| 📋 JSON | 导出当前筛选后的帖子数据，格式化输出 |

> 导出范围：当前搜索和分类筛选后的帖子

---

## 日历视图

- 月历网格布局，7 列（周日 ~ 周六）
- 有帖子的日期显示彩色圆点（对应分类颜色）和帖子数量
- 点击有帖子的日期，下方展开该日帖子列表
- 再次点击同一日期取消选中
- 月份导航按钮切换上下月
- 今日日期有蓝色边框高亮，选中日期蓝色填充高亮
- 非当月日期灰显

---

## 自动刷新

| 方式 | 说明 |
|:-----|:-----|
| ⏱️ 自动轮询 | 页面内置 5 分钟间隔自动数据轮询 |
| 🔄 手动刷新 | 头部刷新按钮（快捷键 `R`），刷新时按钮旋转动画 |
| 🔄 智能暂停 | 页面切到后台暂停轮询，切回前台立即刷新一次 |
| 🔇 静默失败 | 刷新失败时静默处理，不影响当前展示 |

## URL 状态同步

以下状态会同步到 URL hash，支持书签和分享：

| 参数 | 说明 | 示例 |
|:-----|:-----|:-----|
| `q` | 搜索关键词 | `#q=TRAE` |
| `cat` | 当前分类 | `#cat=技巧分享` |
| `sort` | 排序方式 | `#sort=popular` |
| `view` | 视图模式 | `#view=calendar` |
| `theme` | 主题 | `#theme=dark` |

使用 `history.replaceState` 避免历史记录污染。

## 骨架屏加载

页面加载时显示骨架屏，提供更好的用户体验：

- 模拟分类列视图的布局结构
- 渐变闪烁动画提示加载中
- 数据加载完成后平滑过渡到真实内容
- 减少页面加载时的视觉闪烁

## 键盘导航

| 快捷键 | 功能 | | 快捷键 | 功能 |
|:------:|:-----|:-:|:------:|:-----|
| `/` | 聚焦搜索框 | | `D` | 切换亮色/深色主题 |
| `1` | 分类视图 | | `S` | 打开/关闭统计面板 |
| `2` | 卡片视图 | | `R` | 刷新数据 |
| `3` | 日历视图 | | `Esc` | 清除搜索 / 关闭面板 |

- 输入框内按 `Esc` 会清除搜索并失焦
- 所有可交互元素支持 `Tab` 键切换
- 焦点指示器清晰可见（蓝色轮廓）

## 打印优化

- 隐藏工具栏、返回顶部按钮、主题切换、导出按钮、快捷键提示、统计面板
- 静态头部，无阴影
- 分类列避免分页断裂（`break-inside: avoid`）

## 性能优化

| 优化项 | 实现方式 |
|:-------|:---------|
| 懒加载图片 | 卡片视图图片使用 `loading="lazy"` |
| IntersectionObserver | 超过 100 帖子仅渲染可视区域，滚动到底部批量加载 |
| CSS 变量主题 | 统一主题系统，深色模式零重绘 |
| 防抖搜索 | 200ms 防抖减少不必要的渲染 |
| 平滑动画 | CSS 过渡 + `cubic-bezier` 提升体验 |
| 被动滚动监听 | `passive: true` 优化滚动性能 |
